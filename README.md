# 🌐 AI Language Translation Tool

A user-friendly language translation application developed for the CodeAlpha Artificial Intelligence Internship.

The application allows users to enter text, select source and target languages, and receive an instant translation.

## Features

- Automatic source-language detection
- Support for multiple languages
- Simple and responsive web interface
- Real-time text translation
- Copy-friendly translation output
- Download translations as a text file
- Error handling for empty text and connection problems

## Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Translator service
